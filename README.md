# Marking Up a Letter — MDN Challenge

A semantic HTML exercise based on the [MDN Web Docs challenge](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content/Marking_up_a_letter), completed as part of the µLearn Foundation web development track.

## 📄 About the Project

This project involves marking up a plain-text letter from a university research fellow into well-structured, semantic HTML — the kind that could actually be hosted on a university intranet.

The goal was to apply proper HTML structure and text-level semantics, not just make it *look* right, but make it *mean* something to browsers and assistive technologies.

## 🏷️ HTML Concepts Used

- Structural elements: `<address>`, `<h1>`, `<h2>`, `<ul>`, `<ol>`, `<dl>`
- Inline semantics: `<strong>`, `<em>`, `<abbr>`, `<time>`, `<sub>`, `<sup>`
- Quotation elements: `<q>`, `<cite>`
- Document metadata: `<meta charset>`, `<meta name="author">`, `<title>`
- Basic CSS embedded in `<style>` for layout

## 🛠️ Technologies Used
- HTML5
- CSS3
